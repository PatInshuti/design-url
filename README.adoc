== Requirements
  - Install SQLite
  - Install Redis

== In-Memory Cache

To Start Redis DB
```
./redis-server --port 6379
```

== Server

To package your application:
```
./mvnw clean package
```

To run your application:
```
./mvnw clean compile exec:java
```
